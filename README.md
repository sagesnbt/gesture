# Surgical Gesture Annotation UI

## Overview
This project aims to provide a user-friendly interface for annotating surgical gestures in real-time. It is designed to aid researchers and surgeons in studying surgical techniques and enhancing skill training.

## Features
- **Real-time Gesture Tracking**: Capture and analyze gestures as they are performed.
- **Intuitive UI**: A clean and straightforward interface for easy navigation and interaction.
- **Annotation Tools**: A set of tools for detailed gesture annotation and categorization.
- **Data Export**: Option to export annotated data for further analysis.

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/sagesnbt/gesture.git
   cd gesture
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, you can start annotating gestures by following the on-screen instructions. The interface will guide you through the annotation process.
